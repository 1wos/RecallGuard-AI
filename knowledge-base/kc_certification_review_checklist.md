# KC Certification Review Checklist

## Required Vendor Fields

For potentially KC-regulated products, vendor submissions should include:

- Product name
- Model name
- Manufacturer or importer
- KC certification number
- Product category

## Missing Information Handling

If the vendor omits model name or KC certification number, RecallGuard AI should not mark the product as approved. The correct outcome is `REVIEW` with a request for additional evidence.

## Strong Evidence

Strong evidence includes:

- Exact certification number match
- Exact model name and manufacturer match
- Public certification or recall source with publication metadata

## Weak Evidence

Weak evidence includes:

- Product-name-only match
- Similar model name without manufacturer
- Vendor statement without public or internal evidence
